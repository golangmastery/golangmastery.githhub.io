Your First Go Program

Information

This lab will guide you through writing your first Go program and understanding the basic structure of Go code.

Learning Objectives

By the end of this lab, you will be able to:

  • Set up a Go development environment
  • Create and run a simple Go program
  • Understand the basic structure of a Go program
  • Use the fmt package for output

Introduction

Go (or Golang) is a statically typed, compiled programming language designed at Google. It's known for its simplicity, efficiency, and built-in concurrency features. In this lab, we'll write our first Go program and explore the basic structure of Go code.

Setup

1

Set up your environment

First, make sure you have Go installed on your system. You can check by running:

Terminal

go version

If Go is not installed, follow the instructions on the official Go website.

2

Create a new project

Create a new directory for your project:

Terminal

mkdir hello-world cd hello-world

Initialize a new Go module:

Terminal

go mod init example.com/hello-world

This creates a go.mod file that defines your module and its dependencies.

Let's break down this code:

  • package main: Every Go file starts with a package declaration. The main package is special - it defines an executable program rather than a library.
  • import "fmt": This imports the fmt package, which contains functions for formatted I/O.
  • func main(): The main function is the entry point of your program. When you run your program, execution starts here.
  • fmt.Println("Hello, Go!"): This calls the Println function from the fmt package to print text to the console.
4

Run your program

Run your program using the go run command:

Terminal

go run main.go

You should see the output:

Terminal

Hello, Go!

Congratulations! You've just written and run your first Go program.

Building Your Program

5

Build an executable

The go run command compiles and runs your program in one step. To build an executable file that you can run later, use the go build command:

Terminal

go build

This creates an executable file named after your module (in this case, hello-world or hello-world.exe on Windows).

Run the executable:

Terminal

./hello-world # On Unix-like systems hello-world.exe # On Windows

You should see the same output as before:

Terminal

Hello, Go!
